## ✅ 実装完了した機能

### 1. 動体検出システム
- **実装方式**: フレーム差分アルゴリズム
- **特徴**:
  - 前フレームとの比較により動きを検出
  - メモリ効率的（約200KB）
  - リアルタイム処理（1秒間隔）
  - 感度調整可能
- **ファイル**: `src/main.cpp` の `detectPerson()` 関数

### 2. MJPEG動画録画機能
- **録画形式**: Motion JPEG (.mjpeg)
- **録画時間**: 30秒間
- **フレームレート**: 10 FPS
- **保存先**: SDカード `/recordings/` ディレクトリ
- **ファイル命名**: `rec_YYYYMMDD_HHMMSS.mjpeg`
- **機能**:
  - 検出時に自動録画開始
  - フレームカウント表示
  - アラートと並行動作
- **実装関数**: `startRecording()`, `updateRecording()`, `saveFrameToSD()`

### 3. MJPEGライブストリーミング
- **エンドポイント**: `http://[デバイスIP]/stream`
- **プロトコル**: MJPEG over HTTP
- **形式**: `multipart/x-mixed-replace; boundary=frame`
- **視聴方法**:
  - Webブラウザで直接視聴
  - VLCメディアプレイヤー対応
  - リアルタイム配信
- **実装関数**: `handleStream()`

### 4. 強化されたWebインターフェース
- **機能**:
  - システム状態リアルタイム表示
  - ライブプレビュー（2秒自動更新）
  - 日本語UI
  - レスポンシブデザイン
- **表示情報**:
  - 動作状態（待機中/検出中/アラート/録画）
  - アラート状態
  - 録画状態
  - 稼働時間
  - 空きメモリ
- **実装関数**: `handleRoot()`

### 5. 並行処理システム
- **状態管理**: 複数状態の同時処理
  - アラート動作中に録画継続
  - 検出処理と並行して通知送信
- **タイミング制御**:
  - 10秒間のアラート（ブザー + LED）
  - 30秒間の録画
  - 両方が並行動作

## 🚀 使用方法

### 1. 設定
[config.h](include/config.h) を編集：
```cpp
const char* wifi_ssid = "あなたのWiFi名";
const char* wifi_password = "あなたのWiFiパスワード";
const char* line_channel_access_token = "LINEトークン";
const char* line_user_id = "ユーザーID";
```

### 2. ビルドとアップロード
```bash
# PlatformIO CLIの場合
pio run --target upload

# PlatformIO IDE (VS Code)の場合
Ctrl+Alt+U でアップロード
```

### 3. 動作確認
1. シリアルモニター（115200bps）でIPアドレス確認
2. ブラウザで `http://[IPアドレス]/` にアクセス
3. システム状態を確認
4. カメラの前で動いて検出テスト

### 4. ストリーミング視聴
- **ブラウザ**: `http://[IPアドレス]/stream`
- **VLC**: ネットワークストリーム → URLを入力

## 🛠️ トラブルシューティング

### 動体検出が過敏/鈍感な場合
`src/main.cpp` の `detectPerson()` 関数を調整：
```cpp
// 感度調整（数値を大きくすると鈍感になる）
uint32_t diff_threshold = min_size / 20; // デフォルト: 5%
if (diff > 30) { // デフォルト: 30
```

### SDカードエラー
- FAT32フォーマット確認
- 空き容量確認
- カードの抜き差し
- シリアルログで詳細確認

### ストリーミングが見れない
- ファイアウォール設定確認
- 同一ネットワーク接続確認
- VLCで試す

### メモリ不足
`include/config.h` で画質調整：
```cpp
const int camera_frame_size = FRAMESIZE_VGA;  // SVGAから変更
const int camera_jpeg_quality = 15;  // 品質を下げる
```
